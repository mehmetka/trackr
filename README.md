# trackr: Personal library management, reading/working/todo trackings

## Install
1. Clone the repository
2. ``cd trackr/docker``
3. ``docker compose up``
4. ``cd ..``
5. ``composer install``
6. Edit conf.ini.example file as conf.ini
7. Create logs directory on project root.
